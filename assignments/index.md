# Assignments

This page outlines weekly deliverables, submission requirements, and optional R programming extensions.

---

## Weekly Deliverables

Each week you will complete assignments that reinforce skills from tutorials and class exercises.

### Format

Each deliverable typically consists of:
1. One or more screenshots of your Orange workflow/output
2. A short written reflection (1-2 paragraphs)
3. Your `.ows` workflow file (where applicable)

### Submission

- Commit deliverables to your GitHub repository in the appropriate weekly folder (e.g., `week01/`, `week02/`)
- **Deadline:** 17:00 on the Monday following class unless otherwise specified
- Grades are recorded from your GitHub repo (no Brightspace submission needed)

### Grading Scale

| Grade | Meaning |
|-------|---------|
| 2 | Fully complete and accurate |
| 1 | Attempted but incomplete or inaccurate |
| 0 | Incomplete, late, or not attempted |

## GitHub Repository

You are required to maintain a **private GitHub repository** for this course.

### Setup

1. Create a private repo named: `DH-TopicalReading-<Surname>`
2. Add the instructor (**scdenney**) as a collaborator
3. Submit your repo URL to the instructor

### Required Structure

```
DH-TopicalReading-<Surname>/
├── assignments/
│   ├── week01/
│   │   ├── week01-deliverable.md
│   │   └── screenshots/
│   ├── week02/
│   │   └── ...
│   └── final-project/
└── README.md
```

## R Programming (Optional)

Students interested in developing R programming skills can complete optional modules for extra credit.

### Platforms

- **Swirl** - Interactive R tutorials in RStudio
- **DataCamp** - Online courses (institutional access provided)

### Bonus Modules

**Swirl R Programming:**
- Lesson 5: Missing Values
- Lesson 8: Logic
- Lesson 9: Functions

**Swirl Exploratory Data Analysis:**
- Lesson 13: Dimension Reduction

**DataCamp:**
- Data Manipulation & Visualization
- Intermediate R
- Text Mining with Bag-of-Words in R

### Extra Credit Policy

- Up to 0.5 points added to final grade upon completion
- Students who opt in but do not complete may receive a minor penalty
